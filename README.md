## Bug Tracker (Console App)
A compact Java tool for managing software defects with unique IDs and real-time status tracking.
## ✨ Key Features

* Auto-ID: Generates IDs like BUG001.
* Instant Search: Uses HashMap for lightning-fast bug lookups.
* Data Safety: Only RESOLVED bugs can be deleted.
* Stats & Filters: Summary dashboard and severity-based filtering.

## 🛠️ Tech Stack

* Java 8+
* Enums: For strict Severity and Status control.
* HashMap: For efficient $O(1)$ data storage.

## 🚀 Quick Start

   1. Compile: javac BugTracker.java
   2. Run: java BugTracker
   3. Use: Follow the on-screen menu to log, update, or filter bugs.
